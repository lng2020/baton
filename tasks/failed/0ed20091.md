# Set chat-section to fixed 1/3 height of parent

In `frontend/css/style.css`, modify the `.chat-section` rule (around line 554) to use a fixed responsive height of 1/3 of its parent:

1. Change the `.chat-section` rule (lines 554-562) from:
```css
.chat-section {
    background: var(--bg-secondary);
    border-bottom: 1px solid var(--border);
    display: flex;
    flex-direction: column;
    flex-shrink: 0;
    max-height: 45vh;
    min-height: 0;
}
```
to:
```css
.chat-section {
    background: var(--bg-secondary);
    border-bottom: 1px solid var(--border);
    display: flex;
    flex-direction: column;
    flex-shrink: 0;
    flex-grow: 0;
    flex-basis: 33.33%;
    height: 33.33%;
    min-height: 0;
}
```

This removes `max-height: 45vh` and instead uses `height: 33.33%` plus `flex-basis: 33.33%` so it takes exactly 1/3 of the parent `.main-content` height. `flex-shrink: 0` and `flex-grow: 0` keep it fixed at that size. The `.kanban-container` already has `flex: 1` so it will fill the remaining 2/3.

2. Also update `.chat-section.collapsed` (line 564) â€” change from `max-height: none` to `height: auto; flex-basis: auto;` so collapsing still works:
```css
.chat-section.collapsed { height: auto; flex-basis: auto; }
```

The parent `.main-content` (line 120) already uses `display: flex; flex-direction: column; overflow: hidden;` which is correct for percentage-based child heights.
